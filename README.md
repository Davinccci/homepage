# homepage

Davinci 的个人项目主页。

这是一个静态单页网站，当前包含：

- 首页主视觉
- GitHub 跳转入口
- 下滑切换的子项目展示区

## Project Structure

`site/index.html`

- 网站主页面
- 包含页面结构、样式和滚动切换脚本

`Caddyfile`

- 当前站点的 Caddy 配置文件

## Notes

- 项目已初始化为 git 仓库，默认分支为 `main`
- 运行时目录 `caddy_config/` 和 `caddy_data/` 已被 `.gitignore` 忽略
